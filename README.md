## 🎯 プロジェクト概要
楽天ROOMアフィリエイトに特化した売れ筋商品リサーチツール。楽天市場APIから人気商品を取得し、魅力的な紹介文を自動生成してROOM投稿をサポートします。

## ✨ 主な機能

### 1. **カスタムリクエスト検索（NEW！）**
   - 自由なキーワードで商品検索
   - 例：「文房具 可愛い」「冬 あったか」「プレゼント 1000円」
   - 10商品を提案

### 2. **カテゴリ別商品検索**
   - 掃除グッズ / アウトドア / DIYグッズ / 自動車関連
   - おまかせ10選（ランダム）

### 3. **履歴管理機能（NEW！）**
   - 一度紹介した商品は二度と表示しない
   - ローカルストレージに履歴を保存
   - 履歴クリア機能付き

### 4. **キャッチーな紹介文生成（改善！）**
   - **1文目**: 心を引きつけるフック（問いかけ形式）
   - **2文目以降**: 商品の魅力・理由・口コミを詳しく説明
   - 150〜250文字の最適な長さ
   - 絵文字を効果的に配置

   **例文**:
   ```
   他人への愛情よりも自分へのご褒美が優先だよね！ 激落ちくん メラミンスポンジ 大容量100個入はいかがですか？✨💡🎯 5430件以上のレビューで評価4.6を獲得している実力派で、実際に使った人からの満足度も抜群。「安定の美味しさ」「コスパが良すぎる」と口コミでも話題沸騰中。売り切れる前にゲットしておきたい逸品です！
   ```

### 5. **ワンクリックコピー機能**
   - 個別コピー
   - 全て一括コピー

### 6. **商品情報の詳細表示**
   - 商品名・価格
   - レビュー数・評価
   - なぜ売れているか
   - 楽天市場リンク

## 📊 データ構造
現在はデモデータで動作中。楽天市場API連携後は実データに切り替え予定。

### 商品データモデル
```typescript
interface Product {
  name: string           // 商品名
  price: number          // 価格
  url: string            // 商品ページURL
  imageUrl: string       // 画像URL
  reviewCount: number    // レビュー数
  rating: number         // 評価
  category: string       // カテゴリ
  description: string    // AI生成の紹介文
  reason: string         // 人気の理由
}
```

## 📝 使い方ガイド

### 基本的な使い方
1. **カスタムリクエスト**: キーワードを入力して検索
   - 例: 「文房具 可愛い」→ 検索ボタンをクリック
2. **カテゴリ選択**: カテゴリボタンをクリック
3. 売れ筋商品リストが表示される
4. 紹介文をコピー
5. 楽天ROOMに投稿

### 履歴管理
- 一度紹介した商品は自動的に除外されます
- 履歴をクリアしたい場合は「履歴をクリア」ボタンをクリック

## 🔑 楽天市場API設定
実データに切り替えるには、以下の環境変数が必要です：
- `RAKUTEN_APP_ID`: 楽天アプリケーションID（設定済み）
- `RAKUTEN_ACCESS_KEY`: 楽天アクセスキー（設定済み）
- `RAKUTEN_AFFILIATE_ID`: 楽天アフィリエイトID（設定済み）

**注意**: 楽天市場APIの旧バージョン（20170706）を使用中。新バージョン（20220601）への移行は楽天側の安定化を待つ。
